Aula de Progamação Front-End - 16/02

A tag html é utilizada para informar ao navegador que tipo de estrutura esta seendo construída,podendo ter titulos, paragrafos, imagens, links, entre outros. Existem diversas tags do HTML, mas as principais são: title, meta, head, body, footer, e div. Dentro dessas tag, é necessario declarar outras duas tags: head e body.
